# 跨市场套利监控系统 - 用户界面设计规格

## 项目概述
基于 Predict.fun 和 Polymarket 的跨市场套利监控系统，支持实时获取多个预测市场的价格数据，识别套利机会。

## 4. 数据展示格式

### 4.1 价格显示

#### 基础价格格式
```typescript
// 价格格式化函数
formatPrice(price: number, currency: string = 'USD'): string {
  if (price === null || price === undefined) return '--';
  
  // 价格范围 0-1，显示为 $0.00 格式
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency: currency,
    minimumFractionDigits: 2,
    maximumFractionDigits: 2
  }).format(price);
}

// 示例输出
// $0.65
// $1.00
// $0.00
```

#### 价格变化显示
```typescript
// 变化格式化
formatChange(current: number, previous: number): string {
  const change = current - previous;
  const percent = (change / previous) * 100;
  const sign = change >= 0 ? '+' : '';
  
  return `${sign}${change.toFixed(2)} (${sign}${percent.toFixed(1)}%)`;
}

// 示例输出
// +0.05 (+8.3%)  ← 绿色
// -0.03 (-4.5%)  ← 红色
```

// 百分比格式化
formatPercent(value: number, decimals: number = 1): string {
  const sign = value >= 0 ? '+' : '';

}

### 4.4 金额格式化

```typescript
// 交易量格式化（智能单位）
formatVolume(value: number): string {
  if (value >= 1_000_000) {
    return `$${(value / 1_000_000).toFixed(1)}M`;
  } else if (value >= 1_000) {
    return `$${(value / 1_000).toFixed(1)}K`;
  }
  return `$${value.toFixed(0)}`;
}

// 示例输出
// $1.2M
// $850K
// $500
```

### 4.5 时间显示格式

```typescript
// 时间格式化
formatTime(date: Date | string, format: string = 'relative'): string {
  const d = new Date(date);
  const now = new Date();
  const diff = now.getTime() - d.getTime();
  
  if (format === 'relative') {
    // 相对时间
    if (diff < 60_000) return '刚刚';
    if (diff < 3_600_000) return `${Math.floor(diff / 60_000)}分钟前`;
    if (diff < 86_400_000) return `${Math.floor(diff / 3_600_000)}小时前`;
    if (diff < 7 * 86_400_000) return `${Math.floor(diff / 86_400_000)}天前`;
    return d.toLocaleDateString('zh-CN');
  }
  
  // 绝对时间
  return d.toLocaleString('zh-CN', {
    year: 'numeric',
    month: '2-digit',
    day: '2-digit',
    hour: '2-digit',
    minute: '2-digit'
  });
}

// 倒计时显示
formatCountdown(endDate: Date): string {
  const diff = endDate.getTime() - Date.now();
  if (diff <= 0) return '已结束';
  
  const days = Math.floor(diff / 86_400_000);
  const hours = Math.floor((diff % 86_400_000) / 3_600_000);
  
  if (days > 0) return `${days}天 ${hours}小时`;
  return `${hours}小时`;
}

// 示例输出
// 刚刚
// 5分钟前
// 2小时前
// 3天前
// 2024/01/15 14:30
// 5天 3小时 (倒计时)
```

## 附录：设计原则

1. **清晰优先**: 重要信息一目了然，减少认知负担
2. **一致性**: 相同功能使用相同交互和视觉样式
3. **反馈及时**: 用户操作后立即给予视觉反馈
4. **容错性**: 错误操作可撤销，关键操作需确认
5. **效率**: 常用功能快速访问，支持键盘操作
6. **可访问性**: 支持屏幕阅读器，色彩不依赖信息传达
